# 基于Python的深度学习音乐推荐系统 python89

## 项目概述

本项目是一款基于Python开发的深度学习音乐推荐系统，能够实时为用户提供个性化的音乐推荐。系统运用深度学习技术和卷积神经网络，以提升音乐推荐的准确性和个性化水平。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: MySQL
- **核心算法**: 深度学习、卷积神经网络
